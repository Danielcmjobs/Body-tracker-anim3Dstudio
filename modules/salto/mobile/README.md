# Módulo Móvil — Pendiente

Reservado para un cliente móvil nativo que grabe vídeo del salto y lo envíe al backend.

> **Estado:** No iniciado. Actualmente la grabación se realiza directamente desde el navegador del móvil accediendo preferentemente a `https://<IP_local>:8443/salto.html` (ver manual de usuario, sección 6).

> **Acceso recomendado actual:** `https://<IP_local>:8443/salto.html` (con certificado local generado).
>
> **Modo legacy:** `http://<IP_local>:8080/salto.html` solo para compatibilidad en pruebas locales.

### Posibles tecnologías

- React Native / Expo
- Flutter
- PWA con Service Workers
